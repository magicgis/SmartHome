NeedsPlugin Class
Should only be applied to Plugin classed This attribute tells the plugin system, that another plugin is required for this plugin to work. This plugin will not be loaded, if the required plugin is not loaded. At startup, this plugin will wait At runtime, the loading process will be canceled
